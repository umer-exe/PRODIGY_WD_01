# Basic Responsive Landing Page

**Live Demo:** https://umer-exe.github.io/PRODIGY_WD_01/  

A lightweight, responsive landing page built with HTML, CSS, and JavaScript. It includes a fixed navbar that changes style on scroll, a Projects dropdown, a light, dark theme toggle with persistence, and an interactive hero section with a spotlight and gentle 3D tilt on hover.

## Features
* Implemented fixed navbar, transparent at top, becomes solid on scroll.
* Added Projects dropdown, right aligned, spaced from the theme toggle.
* Built theme toggle with localStorage persistence and aria-pressed for accessibility.
* Created interactive hero with cursor spotlight and subtle 3D tilt.
* Included smooth scrolling and interactive button hover states.
* Pure HTML, CSS, JavaScript, no build step, no external JS dependencies.
* Deployed on GitHub Pages, live demo link in the README header.
